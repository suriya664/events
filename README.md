# EventPro 2025 - Event Management Template

Professional event management website template built with pure HTML5, CSS3, and Vanilla JavaScript.

## Features

- ✅ Pure HTML/CSS/JS (No frameworks)
- ✅ CSS Grid & Flexbox layouts
- ✅ Mobile-first responsive design
- ✅ Modern purple/indigo gradient theme
- ✅ Clean, minimalist design
- ✅ Form validation & AJAX
- ✅ Image lazy loading
- ✅ Smooth animations

## Getting Started

1. Open `index.html` in your browser
2. No build process required
3. Customize colors, content, and images

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, Variables)
- Vanilla JavaScript
- jQuery 3.7.1 (CDN)
